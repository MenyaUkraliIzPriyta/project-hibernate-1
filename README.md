# 📦 Project Hibernate 1 - ORM приложение на Hibernate

Проект демонстрирует использование Hibernate ORM для работы с базой данных в многослойном Java-приложении. Реализована полноценная CRUD функциональность с веб-интерфейсом.

## 📋 Описание

Это веб-приложение, построенное на Java с использованием Hibernate для объектно-реляционного отображения (ORM). Проект представляет собой типичное многослойное приложение с четким разделением ответственности и веб-интерфейсом для управления данными.

### Основные возможности:
- 🗄️ Полноценная работа с базой данных через Hibernate
- 📝 CRUD операции над сущностями
- 🎨 Веб-интерфейс для управления данными
- 📊 Связи между таблицами (One-to-Many, Many-to-One)
- 🔍 Запросы через HQL и Criteria API

## ⚙️ Технологии

### Основные:
- **Java 8+**
- **Hibernate ORM** 5.x - объектно-реляционное отображение
- **JPA** (Java Persistence API) - стандарт для ORM
- **MySQL/PostgreSQL/H2** - база данных
- **Maven** - сборка проекта

### Веб-слой:
- **Java Servlets** / **Spring MVC** (зависит от реализации)
- **JSP** / **HTML** - представления
- **CSS** - стилизация
- **JavaScript** - клиентская логика

## 🚀 Установка и запуск

### Требования
- JDK 8 или выше
- Apache Maven 3.6+
- MySQL (или другая БД)
